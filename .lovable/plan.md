

# خطة تحسين اللعبة

## المشاكل الحالية وحلولها

### 1. إصلاح أنيميشن الأزمات (تظهر كل دور)
**المشكلة**: `activeCrisis` تبقى في `gameState` ولا تُمسح بعد العرض. كل re-render يعيد تشغيل الأنيميشن.
**الحل**: في `useGameLogic.ts` - مسح `activeCrisis` بعد 3 ثواني. وفي `CrisisAnimation.tsx` - إضافة مقارنة عبر `useRef` لمنع إعادة التشغيل بنفس الأزمة. عرض الأنيميشن فقط للأحداث `critical` و `high`.

### 2. تحسين نظام القصة
- إضافة شريط تقدم واضح في `StoryPanel.tsx` مع عنوان الفصل الحالي وملخص
- عرض إشعار عند فتح فصل جديد
- ربط الشخصيات بالقصة بشكل أوضح - عرض تغييرات حالة الشخصيات

### 3. تحسين الخريطة التفاعلية
- في `InteractiveMap.tsx`: إضافة حدود أوضح بين المناطق باستخدام SVG paths
- ألوان أكثر تمييزاً حسب حالة المنطقة
- عرض معلومات أكثر عند اختيار منطقة (الموارد، المشاكل، الحاكم)

### 4. نظام أهداف الأدوار (Turn Goals)
- إضافة `turnGoal` في `GameState` - هدف يتغير كل 3-5 أدوار
- أمثلة: "حافظ على الاقتصاد فوق 40"، "اخفض الاضطرابات في الجنوب"
- مكافأة عند تحقيق الهدف، عقوبة عند الفشل
- عرض الهدف الحالي في واجهة اللعبة

### 5. التحكم بالدولة كاملة أو بالمقاطعات
- إضافة وضع "إدارة المقاطعة" عند اختيار منطقة من الخريطة
- قرارات خاصة بكل منطقة (بناء مصنع، تعيين حاكم، إرسال قوات)
- عرض قرارات مختلفة حسب المنطقة المختارة

### 6. زيادة المحتوى (بالعربي)
- إضافة 20+ قرار جديد مع أحداث لاحقة
- إضافة 10+ حدث عشوائي جديد
- توسيع قوالب المولد التلقائي بـ 5+ قوالب إضافية
- إضافة 3 فصول قصة إضافية
